# CONTRIBUTING

Thank you for your interest in contributing to POE-EDU. We welcome improvements that make the project more educational, secure, and useful.

How to contribute
- Fork the repository and open a pull request.
- Keep changes focused and document new features in README.md.
- Add tests for new functionality where possible.
- Do not commit secrets (.env files, tokens).

Issues
- For bugs or feature requests, open an issue and provide a minimal reproduction.

Code style
- Follow standard Python formatting (black / isort recommended).

Security
- If you discover a security issue, please disclose it privately to the maintainers instead of opening a public issue.
